# `vrelease-js`

<img src="icon.svg" height="240px" align="right"/>

Javascript wrapper for [`vrelease`][vrelease]. Install with npm:

```sh
npm install --save-dev vrelease-bin
```

Or yarn:

```sh
yarn add --dev vrelease-bin
```
